Entendendo line-height: 1.6;

Aqui você consegue ver, testar e comparar o espaçamento entre linhas. Em outras palavras: é o “respiro” do texto. Sem isso, o parágrafo vira muvuca visual.

p { font-size: 16px; line-height: 1.6; }

🧠 O que significa?

line-height define a altura de cada linha de texto. Quando você usa 1.6, está dizendo que a altura da linha será 1.6 vezes o tamanho da fonte.

Exemplo: se o texto estiver com 16px, a altura da linha será 25.6px.

Tamanho da fonte 16px
line-height 1.6
Altura da linha 25.6px

Prévia do parágrafo

Este é um exemplo de parágrafo para mostrar como o line-height influencia a leitura. Quando as linhas ficam muito grudadas, o texto parece apertado. Quando o espaçamento está equilibrado, a leitura fica mais confortável e organizada.

💡 Dica: 1.5 a 1.8 costuma funcionar muito bem para parágrafos.

📊 Comparação rápida

1.0 — apertado

Este texto está com pouco espaço entre as linhas. Dá para ler, mas cansa mais e parece tudo espremido.

1.6 — confortável

Este texto está com um espaçamento mais equilibrado. A leitura fica mais leve, organizada e agradável.

2.3 — exagerado

Este texto tem espaço demais entre as linhas. Em alguns casos pode funcionar, mas normalmente fica solto demais.

🎯 Resumo rápido

  • line-height controla a altura entre as linhas do texto.
  • 1.6 significa: 1.6 vezes o tamanho da fonte.
  • Valores entre 1.5 e 1.8 costumam ficar ótimos em parágrafos.
  • Títulos costumam usar valores menores, como 1.1 ou 1.2.
