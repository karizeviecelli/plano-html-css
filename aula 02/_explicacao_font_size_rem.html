CSS Fundamentos

Entendendo font-size: .9rem;

Essa linha de CSS parece simples, mas ela ajuda a deixar o texto mais organizado, mais bonito e mais profissional. Traduzindo sem enrolação: ela define o tamanho da fonte como 90% do tamanho base do site.

O que é font-size?

É a propriedade do CSS que controla o tamanho da letra. Quando você muda o font-size, o texto pode ficar maior, menor, mais confortável ou mais difícil de ler. O CSS não perdoa bagunça visual.

p { font-size: 16px; }

O que significa .9rem?

A unidade rem é baseada no tamanho do texto do elemento raiz, normalmente o <html>. Se o padrão for 16px, então:

  • 1rem = 16px
  • .9rem = 14.4px

Ou seja: o texto fica um pouquinho menor que o normal.

Exemplo 1: comparação simples

body { font-size: 1rem; } p { font-size: .9rem; }

Explicação: o texto do parágrafo ficará levemente menor que o tamanho padrão do site.

Exemplo 2: hierarquia visual

h1 { font-size: 2rem; } p { font-size: 1rem; } small { font-size: .9rem; }

Explicação: o título chama mais atenção, o texto principal fica no tamanho padrão e os detalhes aparecem menores.

Por que usar rem?

  • Ajuda na responsividade
  • Melhora a acessibilidade
  • Mantém proporção entre os textos
  • Facilita ajustes no projeto inteiro

Por que não usar só px?

O px funciona, mas é uma medida fixa. Já o rem é mais flexível. Ele acompanha melhor mudanças de tamanho no projeto. Em outras palavras: menos dor de cabeça, mais organização.

Exemplo 3: teste prático

Veja como o rem reage quando o tamanho base muda:

html { font-size: 20px; } p { font-size: .9rem; }

Agora o parágrafo não terá mais 14.4px. Ele passará a ter 18px, porque .9 × 20 = 18. É aí que mora a mágica do rem.

Dica de professora esperta: peça para os alunos mudarem o valor do html e observarem todo o layout. É um ótimo jeito de mostrar que o rem é relativo.

Resumo rápido

  • font-size define o tamanho do texto
  • rem usa como base o tamanho do <html>
  • .9rem significa 90% do tamanho base
  • É ótimo para detalhes, descrições e hierarquia visual
