# plano-html-css

 GitHub Pages 
O GitHub Pages é gratuito, permite personalizar o domínio e é amplamente usado por estudantes e profissionais de desenvolvimento. Adamsilver

Crie uma conta em github.com
Clique em New repository → nome: plano-html-css
Marque "Add a README file" → clique em Create repository
Clique em "uploading an existing file" → arraste o plano_html_css_senai.html → renomeie para index.html → clique em Commit changes
Vá em Settings → Pages → Branch: main → Save
Em ~1 minuto o site estará em https://seu-usuario.github.io/plano-html-css

Vantagem: link permanente, profissional, e você pode atualizar o arquivo quando quiser.
